# 画像生成プロンプト集

## thumbnail.png「サムネイル（アイキャッチ）」

```
【サイズ】1200×630px（OGP標準）
【背景】クリーンなテックブログ風。ベースは白〜明るいグレー、アクセントにシアン（#06B6D4）とピンク（#EC4899）を少量使う。暗くなりすぎない、明るく読みやすい印象にする。

【レイアウト】
- 中央：動画編集ソフトのプレビュー画面のような大きなキャンバスを置き、その中にキャラクターが解説している様子を入れる。
- 上部〜中央：大きな見出しテキスト「Remotion × MotionPNGTuber」を、キャンバス上のタイトルテロップのように配置する。縮小表示でも読めるように、横幅の60〜75%程度を使ってかなり大きく入れる。
- 見出しの近く：小さめの補助テキスト「キャラクターに解説させる」
- 下部：Remotionのタイムライン、音声波形、字幕トラック、`mouth_track.json` の座標点を編集UIとして配置する。
- 右側にキャラクター、左側にタイトルを置く定番構図にはしない。

【スタイル】
- フラットで読みやすい技術記事サムネイル
- 文字は太めのゴシック体
- キャラクターよりも「Remotion × MotionPNGTuber」の文字を主役にする
- タイトル文字を小さくしない。サムネイル内で最も目立つ要素にする
- 余白を広めに取り、ブログ一覧で縮小されても読めるようにする

【制約】
- 実在サービスのロゴをそのまま描かない
- テキストは端から100px以上内側に配置
- 派手なグラデーション背景や過度な装飾を避ける
- 全体を暗くしすぎず、ブログ一覧で見たときに軽く明るい印象にする
```

# 画像生成プロンプト集

## thumbnail.png「サムネイル（アイキャッチ）」

```
【サイズ】1200×630px（OGP標準）
【用途】ブログ記事「Stack CoreS3向けの表情素材を作るツールを作りました」のサムネイル

【背景】
- ダークネイビーからティールへの落ち着いたグラデーション
- クリーンなテックブログ風
- 右上に薄いシアンの円形アクセント

【レイアウト】
- 左側に濃い半透明カードを置く
- カード内に大きく「StackFace Maker」
- 補助テキストとして「Stack CoreS3」「Local Web App」「Eye / Mouth Assets」
- 右側にStackFace Makerのマスク調整画面スクリーンショットを配置
- スクショは青い枠で囲み、ツール画面であることが一目で分かるようにする

【制約】
- ブログ一覧で縮小されても「StackFace Maker」が読める
- キャラクター絵を主役にしすぎず、ツール紹介のサムネとして見せる
- 実在ロゴは使わない
```

## 01-create.png「作成画面のスクリーンショット」

```
【画像タイプ】実スクリーンショット
【用途】StackFace Makerで標準表情、感情、反対状態画像を指定する画面を説明する
【撮影内容】
- 作成タブ
- 標準表情のプレビュー
- 追加プロンプト欄
- 基準の目・口状態
- 反対状態画像のプレビュー
【制約】本文中でUIの流れが分かるように、画面全体を1280x720で撮影する
```

## 02-adjust-mask.png「マスク調整画面のスクリーンショット」

```
【画像タイプ】実スクリーンショット
【用途】目・口マスク、ベース透過、4パターンプレビューを説明する
【撮影内容】
- 調整・保存タブ
- ベース透過スライダー
- 目パーツと口パーツの赤いマスク
- 右側の2x2プレビュー
- 自動反映の状態表示
【制約】マスクの赤い半透明表示とプレビューが同時に見える構図にする
```

## 03-output-faces.png「作成した表情素材一覧」

```
【画像タイプ】実素材を並べたコンタクトシート
【用途】複数感情 × 目口ON/OFFの出力例を見せる
【内容】
- Neutral / Happy / Sad / Angry / Thinking の5感情
- Eye OFF / Eye ON と Mouth OFF / Mouth ON の4状態
- 320x240 PNGとして出力される顔素材
【制約】サンプル画像として見やすいように、行と列の意味が分かる見出しを入れる
```
